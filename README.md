# advent-of-code-2025

## Execution

```
cargo run -- -h

cargo run -- --day 1 --part 1
```

## Usage

```
Usage: advent-of-code-2025 [OPTIONS] --day <DAY> --part <PART>

Options:
      --day <DAY>    Run solution for this day
      --part <PART>  Run solution for this part
      --demo         Use demo input
  -h, --help         Print help
```
